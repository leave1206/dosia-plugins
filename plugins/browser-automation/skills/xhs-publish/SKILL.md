# xhs-publish — 小红书笔记发布

委派 **browser-operator** sub-agent，传入标题、正文、图片路径。

## 参数解析

- **标题**: 必填，不超过 20 字
- **正文**: 必填，可包含 #话题 标签
- **图片**: 必填，至少 1 张，本地文件路径列表
- **话题**: 从正文中提取 #xxx 标签，或用户单独指定

## 输入校验

```
如果没有图片:
  → 返回错误: "小红书笔记必须包含至少 1 张图片"

如果标题 > 20 字:
  → 返回提示: "标题超过 20 字限制，请缩短"
```

## 执行流程

### Step 1: 打开创作者中心

```
navigate_page → https://creator.xiaohongshu.com/publish/publish
wait_for → 发布页面加载完成
```

未登录:
```
take_snapshot → 检查是否跳转到登录页
→ 返回错误: "请在 Chrome 中打开 creator.xiaohongshu.com 登录后重试"
```

### Step 2: 上传图片

```
对每张图片:
  upload_file → 选择本地图片文件
  wait_for → 上传进度完成 / 图片缩略图出现
```

确认所有图片上传成功:
```
take_snapshot → 检查已上传图片数量是否正确
```

如果上传失败，记录失败文件并继续其他图片。

### Step 3: 填写标题

```
click → 标题输入框
fill → 输入标题文本
```

### Step 4: 填写正文

```
click → 正文编辑区
type_text → 输入正文内容
```

正文中的 #话题 标签，小红书编辑器会自动识别为话题链接。

如果需要手动添加话题:
```
click → 话题/标签输入区域（"# 添加话题"）
type_text → 输入话题关键词
wait_for → 话题建议列表出现
click → 选择匹配的话题
```

### Step 5: 预览确认

```
take_screenshot → 截取完整发布页面（图片 + 标题 + 正文）
```

**返回截图给主 Agent: "笔记已编辑完毕（N 张图片），请确认后回复'发布'。"**

**此处停止，等待确认。**

### Step 6: 发布

```
click → "发布" 按钮
wait_for → 发布成功提示 / 跳转到已发布页面
take_screenshot → 截取成功状态
```

返回发布状态。

## 常见异常

| 异常 | 处理 |
|------|------|
| 未登录 | 返回提示登录 |
| 图片上传失败 | 报告失败文件，继续其他图片 |
| 图片格式不支持 | 提示支持的格式（jpg/png/webp） |
| 标题超长 | 返回提示缩短 |
| 发布按钮不可点击 | take_snapshot 检查原因（可能缺少必填项） |
| 内容审核拦截 | 截图返回审核提示信息 |
